# NutriGenius 

NutriGenius began as a capstone project aimed at addressing Indonesia's childhood stunting crisis, which affects approximately 27.7% of the country's toddler population. The original team reached ~50% completion, establishing foundations in machine learning, mobile development, and cloud infrastructure.

As a passionate developer committed to making a proof of concept on this project to bring it to completion.

# Original Project 

NutriGenius is an innovative mobile application designed to combat childhood stunting through intelligent nutrition monitoring and personalized recommendations.

## 🌟 Features

- **Smart Food Recognition**: Automatically identify food types and nutritional composition through photo capture
- **Growth Monitoring**: ML-powered tracking of child growth metrics
- **Personalized Recommendations**: AI-driven nutritional advice tailored to each child
- **Healthcare Integration**: Seamless connection with healthcare providers and systems
- **Educational Content**: Customized nutritional education materials
